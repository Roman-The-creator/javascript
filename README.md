# JavaScript Linter 
Линтер для JavaScript. Поддерживает лексический, синтаксический и логический анализ кода.

## Основные возможности:
- **Лексер:** Токенизация JS кода с поддержкой комментариев и строк.
- **Парсер:** Построение AST с механизмом восстановления после ошибок.
- **Правила:** Проверка naming-conventions (camelCase), отступов и пустых строк.
- **Анализ логики:** Подсчет цикломатической сложности и поиск неиспользуемых переменных.
- **Autofix:** Автоматическое исправление ошибок форматирования.
